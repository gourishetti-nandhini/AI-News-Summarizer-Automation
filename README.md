# AI News Automation using n8n

## 🚀 Overview
This project is an AI-powered automation workflow built using n8n that collects, processes, and delivers daily AI news updates automatically.

## ⚙️ Features
- Fetches AI news from multiple RSS feeds
- Retrieves event data using APIs
- Uses Google Gemini AI to generate summarized reports
- Sends automated email newsletters

## 🛠️ Tech Stack
- n8n (Workflow Automation)
- Google Gemini API (AI Summarization)
- RSS Feeds & HTTP APIs
- Gmail Integration

## 📌 How It Works
1. A scheduled trigger runs the workflow daily  
2. News and event data are collected from multiple sources  
3. Data is merged and processed  
4. AI generates a structured news summary  
5. The summary is sent via email automatically  

## 📂 File
- automation-workflow.json

## 🎯 Use Case
This automation reduces manual effort by providing a daily AI news summary and keeps users updated with the latest trends and events.
